The Telefunc Transformer prepares your `.telefunc.js` files for:

- The client-side, replacing `.telefunc.js` imports with a thin HTTP client.
- The server-side, for <Link text="automatic TypeScript runtime validation" href="/shield#typescript" doNotInferSectionTitle={true} />.

> It's possible to <b>use Telefunc without transformer</b>, see [#52](https://github.com/brillout/telefunc/issues/52).
