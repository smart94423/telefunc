**Environment**: Server.

<Warning>This is a beta feature.</Warning>

If you don't install Telefunc on your server-side (see <Link href="/transformer" />), then you need to use `config.telefuncFiles` in order to provide the list of `.telefunc.js` files.

```js
// Environment: Server

import { config } from 'telefunc'

// List of telefunc files
config.telefuncFiles = [
  require.resolve('./hello.telefunc.mjs')
]

// The root directory of the project
config.root = __dirname
```

You will also need to set <Link href="/root">`config.root`</Link>.
