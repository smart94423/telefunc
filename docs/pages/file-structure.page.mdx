We recommend to collocate `.telefunc.js` next to component files.

```
TodoItem.telefunc.js
TodoItem.{jsx|vue|svelte|...}
```

Where `TodoItem.telefunc.js` holds the telefunctions needed by the `<TodoItem />` component defined in `TodoItem.{jsx|vue|svelte|...}`.

We talk more about this approach at <Link href="/tour#inversion-of-control" />.
